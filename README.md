## 🚀 How to Run

### Option 1 — Direct (simplest)
Double-click `index.html` → opens in your default browser

### Option 2 — VS Code Live Server (recommended)
1. Open the `codenpixel` folder in VS Code
2. Install **Live Server** extension
3. Right-click `index.html` → **Open with Live Server**
4. Site runs at `http://127.0.0.1:5500`

### Option 3 — Python local server
```bash
cd codenpixel
python -m http.server 5500
# Open: http://localhost:5500
```

## 🎨 Tech Stack
- **HTML5** — Semantic structure
- **CSS3** — Custom properties, glassmorphism, animations
- **Vanilla JS** — Canvas 3D, custom cursor, modal system
- **GSAP 3** — Scroll-triggered animations, stagger effects
- **Lenis** — Silky smooth momentum scrolling
- **Swiper.js** — Touch-friendly slider support

## 🌗 Theme
- **Light mode** — Warm cream/yellow backgrounds with neon pink accents
- **Dark mode** — Deep void black with glowing neon pink & cyber yellow
- Toggle the 🌙 button in the top nav to switch
